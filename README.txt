БРАТИШКА, перед тем как запустить скрипт сделай некст степы:

1. сначала закинь свои профили в "profiles.json"

Как закинуть:
{
    "profile_id": "mnemonic_phrase",
}


2. получи тестовые токены обязательно блять, скрипт не делает этой хуйни
3. по необходимости обнови проект
Как обновить:
    1. Открой командную строку
    2. Зайди в директорию с проектом
    3. Введи команду - "./update_script.sh"

4. МОЖЕШЬ ЗАПУСКАТЬ, СУЧАРА
